## tips ##
use sublime text3 plugin to run the scripts
> 
> 1. install nodejs
> 
> 2. in the editor'menu, you can tool->Build System->New Build System
> 
> 3. copy teh code into a new file:
```
{
    "cmd": ["node", "$file"],
    "selector": "source.js"
}

```

> 4. ctrl + B to run scripts
